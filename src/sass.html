Att använda SASS

Jag upplevde att det var krångligare i början än att använda vanlig CSS. Jag kan se hur SASS är till stor hjälp i större projekt, för att strukturera upp all kod, men i mindre projekt blir det lite krystat att hitta användningsområden till alla olika filer. För mig kan det också bli förvirrande med partials, då jag funderar - hör länkar hemma under layout (där huvudmenyn ligger) eller under components där länkar också kan ligga?

Anna Danielsson, Webbutvecklingsprogrammet
