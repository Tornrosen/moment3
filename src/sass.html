Att använda SASS

Jag upplevde att det var krångligare i början än att använda vanlig CSS. Jag kan se hur SASS är till stor hjälp i större projekt, för att strukturera upp all kod, men i mindre projekt blir det lite krystat att hitta användningsområden till alla olika filer. För mig kan det också bli förvirrande med partials, då jag till exempel funderar på - hör länkar hemma under layout (där huvudmenyn ligger) eller under components där länkar också kan ligga? Jag upplevde att color functions var lite svåra att få till och hade problem med att fundera ut vad jag skulle använda en if/else-sats till. Dark och Light-mode som har funkat bra förut krånglade också.

Anna Danielsson, Webbutvecklingsprogrammet
